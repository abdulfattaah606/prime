# Simple Blog (HTML + CSS + JS)

## Run
- Open `index.html` in your browser (double-click it).

## What you get
- Home page with a list of posts
- Post detail page
- “New post” editor

## Routes
- `#/` home
- `#/new` new post
- `#/post/<id>` view a post

## Storage
- Posts are saved to your browser via `localStorage`.
- Use the **Reset demo** button in the footer to restore the sample posts.
